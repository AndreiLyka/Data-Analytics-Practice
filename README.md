# Coffe_sales_analysis_project


## Project Overview
This project focuses on analyzing a company's sales data. The goal is to identify top-selling products, regions with the highest revenue, seasonal trends, and provide actionable business recommendations.

## Project Structure
- `data/` — CSV source data
- `notebooks/` — Jupyter Notebook containing analysis
- `reports/` — charts and visualizations
- `requirements.txt` — required Python libraries

## Tools Used
- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook

## Results
The notebook includes visualizations for top products, regional and monthly sales analysis, and business insights derived from the data.
